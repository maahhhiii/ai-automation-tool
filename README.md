# 🤖 AI Automation Toolkit

A Python-based AI automation project powered by Gemini API.

## Features

- ✉️ AI Email Writer
- 📄 File Summarizer
- 💻 Code Explainer
- 📑 Resume Analyzer
- 📚 PDF Merger
- 🌐 Web Scraper + AI Summary
- 📝 History Saving

## Tech Stack

- Python
- Gemini API
- BeautifulSoup
- Requests
- PyPDF2
